A full-stack web application designed to simplify hotel room reservations and management. The system allows users to browse available rooms, make bookings, and manage their reservations, while administrators can handle hotel listings, room details, and customer data.

 Features:

User registration and login with secure authentication
Search and filter hotels based on preferences
Real-time room availability checking
Booking confirmation and cancellation options
Admin dashboard for managing rooms, hotels, and users
Responsive frontend built with modern UI components
